#    → Or use different port: uvicorn app:app --reload --port 8001
#    → Update the fetch URL in ChatWindow.jsx if you change the port

# ❌ Voice triage says "Triage currently unavailable"
#    → Check ml-service/.env has the real GROQ_API_KEY (not placeholder)
#    → Check FastAPI terminal for errors
#    → Restart: uvicorn app:app --reload --port 8000

# ❌ Mic button doesn't work
#    → Must use Chrome or Edge (Firefox doesn't support Web Speech API)
#    → Must be on localhost (HTTPS required for non-localhost)
#    → Check browser mic permissions

# ❌ npm install fails
#    → Delete node_modules and package-lock.json, retry:
#      Remove-Item -Recurse -Force node_modules, package-lock.json
#      npm install

#    → Ensure you're in the venv: .\venv\Scripts\Activate.ps1
#    → Try: pip install --upgrade pip && pip install -r requirements.txt
